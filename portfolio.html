Bank Website

This website I created is a mock banking simulator that allows you to create and log in to an account, create bank checking and saving accounts, transfer money, and add friends to transfer money to them too. It is on Github for viewing of the code.

Arcade Machine

This is a fully functioning aracde machine. I constructed the shell of the machine using my woodworking knowledge and made holes and slots for functioning buttons. I installed a Raspberry Pi which emulates just under 1000 classic games and wired the buttons to the Raspberry Pi.

CNC Engraving

My old teacher asked me how to work the CNC engraving machine when it was a new addition to the shop and no one knew how to use it. I ended up making a laminated step-by-step guide to use the machine and this was the first test piece.
